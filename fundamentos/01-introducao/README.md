# Introdução ao SQL

Este módulo apresenta os **fundamentos do SQL**, explicando o que é,
para que serve e como ele é utilizado no dia a dia de ambientes reais
de dados e sistemas.

Aqui o foco não é apenas executar comandos, mas **entender os conceitos**
que sustentam o uso do SQL, preparando a base necessária para consultas,
análises e operações mais avançadas nos próximos módulos.

## O que você vai aprender neste módulo

- O que é SQL e por que ele é importante
- Diferença entre banco de dados e sistema gerenciador de banco de dados (SGBD)
- O que são tabelas, colunas e registros
- Tipos básicos de comandos SQL
- Como o SQL é utilizado em ambientes de trabalho

## Estrutura do módulo

- `conceitos.md` → Explicação conceitual dos principais termos
- `introducao-ao-sql.md` → Visão prática e contextual do SQL
- (Nos próximos módulos: exemplos, exercícios e consultas reais)

## Pré-requisitos

Nenhum. Este módulo é indicado para quem está iniciando
ou deseja reforçar conceitos fundamentais.
